📦 MINDESTANFORDERUNG:
   ✅ Nur install.php hochladen
   ✅ Das war's!

───────────────────────────────────────────────────────────

🚀 SCHRITT-FÜR-SCHRITT:

1. FTP-Programm öffnen (z.B. FileZilla)
   └─ Server: ftp.deine-domain.de

2. Navigiere zu:
   └─ /public_html/ (cPanel)
   └─ /httpdocs/ (Plesk)

3. Dateien hochladen:
   ✅ PFLICHT: install.php
   ✅ Optional: .htaccess
   ✅ Optional: logs/ Ordner
   ✅ Optional: database_complete.sql (Fallback)

4. Browser öffnen:
   └─ https://deine-domain.de/install.php

5. Installation folgen:
   ├─ Dateien werden automatisch von GitHub geladen
   ├─ Datenbank-Zugangsdaten eingeben
   ├─ Admin-Account erstellen
   └─ Fertig! 🎉

───────────────────────────────────────────────────────────

💡 WICHTIG:

   ⚠️  Du musst NICHT alle Dateien hochladen!
   ⚠️  install.php lädt automatisch von GitHub!
   ⚠️  Nur bei Problemen: Alle Dateien manuell hochladen
